# 🤖 AlphaBot - Basic Chatbot

## Description
AlphaBot is a simple rule-based chatbot developed using Python.

## Features
- Greets users
- Answers simple questions
- Responds to predefined commands
- Exits using "bye"

## Technologies Used
- Python
- if-elif statements
- Loops
- Input/Output
